# Covid Nepal Data

    This App is a tracker for Coronavirus . It Gives report for worldwide data or data of Nepal.

# Features
    -> Integration of RIVE
    -> Bear Animation available offline
    -> Follows BLoC Pattern

Data Source -> [Api](https://api.covid19api.com)
